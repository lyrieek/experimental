报这个错的原因是因为在Action里面新增了一个需要返回json的方法，

@Action(value = "", results = {@Result(name = "success", type = "json", params = { "root", "result"})})

而之前Action里面没返回json的方法，而status2返回json需要在类里面加一个注解

@ParentPackage("json-default")

所以解决办法就是加上面那个注解
